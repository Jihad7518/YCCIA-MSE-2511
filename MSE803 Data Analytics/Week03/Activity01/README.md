# Correlation Analysis: Age vs Net Worth

## Objective

To calculate and visualize the correlation between Age and Net Worth.

## Method

* Loaded dataset using pandas
* Cleaned column names
* Computed Pearson correlation
* Visualized using scatter and regression plots

## Result

Correlation coefficient: **0.882**

## Interpretation

There is a strong positive relationship between age and net worth. As age increases, net worth generally increases, though with some variation.

## Conclusion

The analysis indicates a strong positive correlation, suggesting wealth tends to accumulate over time but is influenced by other factors.
