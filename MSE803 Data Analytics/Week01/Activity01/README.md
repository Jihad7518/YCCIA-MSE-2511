# Week 1 Activity 1 - Data Analytics

## Objective

Perform descriptive analysis using data aggregation.

## Dataset

The dataset contains aggregated sales and quantity data for categories A, B, and C.

## Analysis

The dataset was already aggregated, so no further calculations were needed. The data was analyzed by comparing totals and averages.

## Results

* Category B has the highest sales, average sales, and quantity sold.
* Category A has the lowest performance.
* Category C shows moderate performance.

## Conclusion

Category B is the best performing category, while Category A may need improvement.
